Recognition Ecosystem Journey Map

Employee recognition journey across the flow of work

A horizontal product journey map showing how recognition is discovered, sent, celebrated, remembered, and translated into organizational insight.

Journey Stages
Current State
Future Experience
1 Discovery

Discover Contributions

Help employees identify meaningful contributors while the work is still fresh.

Current Challenges

  • Contributors are forgotten over time.
  • Behind-the-scenes work often goes unnoticed.
  • Recognition is often delayed—or never sent.

Opportunity

Surface recognition opportunities directly within employees' daily workflow using workflow context and AI recommendations.

Trigger

  • Project completed
  • Meeting ended
  • Milestone reached
  • Task completed
  • Cross-team collaboration finished
  • Service anniversary
  • Mentorship completed

User Actions

  • Review suggested contributors
  • Validate AI recommendations
  • Select recipient(s)
  • Review suggested recognition reasons
  • Continue to Create Recognition

System Actions

  • Detect collaborators
  • Analyze work relationships
  • Recommend recipients
  • Generate recognition reasons
  • Suggest recognition categories

Key Outputs

  • Candidate recipients
  • Suggested recognition reasons
  • Suggested recognition category
  • Pre-filled recognition draft
2 Creation

Create Recognition

Help employees recognize others quickly without interrupting their workflow.

Current Challenges

  • Employees hesitate to send recognition.
  • Writing meaningful recognition takes effort.
  • Recognition is often postponed or abandoned.

Opportunity

Reduce the effort of sending recognition using workflow context, AI-assisted drafting, and pre-filled information.

Trigger

  • Recognition opportunity accepted
  • Recognition flow launched
  • Recipient(s) confirmed

User Actions

  • Review drafted message
  • Personalize recognition
  • Choose visibility
  • Confirm category
  • Submit recognition

System Actions

  • Pre-fill recipient information
  • Generate recognition draft
  • Recommend categories
  • Suggest visibility options
  • Capture recognition metadata

Key Outputs

  • Recognition submitted
  • Recognition message
  • Recognition metadata
  • Visibility settings
  • Points (if applicable)
3 Celebration

Celebrate Recognition

Feel acknowledged and share the recognition moment with the team.

Current Challenges

  • Recognition can get lost in notifications
  • Recipients may not know how to respond
  • Public celebration can feel performative

Opportunity

Make recognition visible in the right team context with lightweight celebration actions.

Trigger

  • Recognition submitted
  • Notification delivered

User Actions

  • Open notification
  • Read recognition
  • Reply or react
  • Participate in celebration

System Actions

  • Render recognition card
  • Post to collaboration channel
  • Capture reactions and replies

Key Outputs

  • Visible celebration
  • Recipient response
  • Team participation signal
Optional

Redeem Points

Understand available points and redeem them for meaningful rewards.

Current Challenges

  • Users may not realize recognition points can be redeemed
  • Points and recognition can feel disconnected
  • Redemption is transactional and may distract from the moment

Opportunity

Use Microsoft surfaces to make point redemption discoverable, while keeping marketplace browsing and fulfillment in Workhuman.

Trigger

  • Recognition with points received
  • Points balance updated
  • Reminder to redeem unused points

User Actions

  • View points balance
  • Open Workhuman Marketplace
  • Browse available rewards
  • Select a reward
  • Complete redemption in Workhuman

System Actions

  • Show points received in the recognition notification
  • Provide deep link to Workhuman Marketplace
  • Sync recognition and points balance where available
  • Keep reward fulfillment in Workhuman

Key Outputs

  • Points balance awareness
  • Workhuman marketplace visit
  • Reward redemption
  • Fulfillment status
4 Memory

Build Career Visibility

Understand and demonstrate impact over time.

Current Challenges

  • Recognition becomes hard to find later
  • Impact evidence is scattered
  • Managers may miss talent signals

Opportunity

Turn recognition history into career evidence and lightweight talent insight.

Trigger

  • Recognition record created
  • Multiple recognition moments accumulate over time

User Actions

  • Review recognition history
  • Explore strength themes
  • Select evidence for career discussion
  • Review team recognition patterns

System Actions

  • Aggregate recognition records
  • Extract strength themes
  • Generate impact summaries
  • Surface talent signals

Key Outputs

  • Impact summary
  • Strength profile
  • Career evidence
  • Talent signal
